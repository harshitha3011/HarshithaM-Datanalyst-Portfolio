**Project Overview**
This repository showcases my complete end-to-end data analytics workflow, starting from raw data processing to generating business insights and statistical validation.
The project is divided into five structured tasks, each focusing on a critical stage of the data analytics lifecycle

**Task 1: Data Immersion & Wrangling**
In this task, I worked on understanding and cleaning the raw dataset (Customers.csv). I handled missing values, removed duplicates, standardized data formats, and transformed the dataset into a clean and structured format (customers_cleaned.csv) ready for analysis.

**Task 2: Exploratory Data Analysis & Business Insights**
This phase involved performing univariate and bivariate analysis to uncover patterns and trends in the data. I used visualizations and aggregation techniques to answer key business questions and generate meaningful insights about customer behavior.

**Task 3: Deep-Dive Analysis & Dashboarding**
In this task, I defined key performance indicators (KPIs) and performed customer segmentation using clustering techniques (K-Means). The processed data was structured into a dashboard-ready format and used to support business decision-making through visualization tools.

**Task 4: Data Storytelling & Statistical Validation**
This phase focused on converting analytical findings into a clear business narrative. I performed hypothesis testing using statistical methods (ANOVA) to validate insights and support conclusions with data-driven evidence.

**Task 5: Capstone Integration**
All tasks were consolidated into a structured portfolio. A final presentation was created summarizing the complete workflow, insights, and learnings from the internship.
